BIOS:
Cerca un eseguibile nell'MBR (primi 512 B) che si trova nel primo settore del disco principale

UEFI:
Non ci sono solamente i dispositivi fisici da cui poter fare il boot, ma vediamo una voce per ogni eseguibile .efi installato.
Viene poi avviato un programma .efi nella partizione fat32 (bootloader).

-> BIOS 
-> BOOTLOADER GRUB(programma minimale che carica tutto quello che server per avviare l'OS): caricato initial run disk, disco iniziale che contiene driver indispensabili per montare la partizione del SO e i driver input output base come tastiera, mouse e driver grafici. 
-> KERNEL(cuore sistema operativo che mette in comunicazione software e hardware cosi che gli applicativi non debbano dipendere dalle specifiche hardware).
con uname -r vedo la versione del kernel.
-> INIT SYSTEM (meta programma che serve a lanciare tutti i servizi che rendono operativo il nostro pc: servizio di networking, servizio ssh e altri servizi in background in fase di avvio).
per sapere quale sistema di init: /sbin/init --version
-> DISPLAY MANAGER: schermata di login
ls -l /etc/systemd/system/display-manager.service per sapere quela display manager sto utilizzando
Il controllo di username e password viene fatto tramite file /etc/passwd e /etc/shadow
-> DESKTOP ENVIRONMENT: per vedere quale sto usando -> echo $DESKTOP_SESSION

screenfetch/neofetch: sommario informazioni generali sul sistema
